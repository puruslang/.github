### Write code without the Shift key.

**Purus** — _/ˈpuː.rus/_ means _pure_ in Latin — is a beautiful, simple, and easy-to-use language that compiles to JavaScript.

</div>

```purus
const greeting be //;Hello, [name]!;//
console.log[greeting]
```
